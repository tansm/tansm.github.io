# 关于扩展Visual Studio 2005的相关资料

> 原文：https://www.cnblogs.com/tansm/archive/2007/03/29/693138.html

公司开发的需要，我们设计的实体设计器需要集成到VS中，并自动创建强类型的源代码。还要做编译时的各种检查，所以不得不看一些资料，来知道如何扩展VS2005。

需要下载 Visual Studio 2005 SDK Version 4.0

http://www.microsoft.com/downloads/details.aspx?FamilyID=51a5c65b-c020-4e08-8ac0-3eb9c06996f4&displaylang=en

这个SDK包中包含了非常重要的DSL功能，即面向问题域（谁翻译的）的开发语言。
